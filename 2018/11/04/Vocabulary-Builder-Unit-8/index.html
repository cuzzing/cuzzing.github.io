Vocabulary Builder Unit 8

Vocabulary Builder Unit 8

PORT = to carry

  1. portage [两条水路间的] 陆上运输
    • overland 经由陆地的
    • canoe 独木舟，小船
    • none other than 不是别的而正是
  2. portfolio 公文包；投资；作品集
    • schlep 携带
    • showcase 陈列柜；展示场合
  3. comport 同意；举止
    • a wedding reception 婚宴
    • dignified 庄严的
    • be a credit to 是······的骄傲，赞扬
  4. deportment 仪态，行为
    • poise 平衡

PEND = to hang / to weigh

  1. pendant
    • ornament 装饰物
    • filigree 精工制品
    • amulet 护身符
  2. append
  3. appendage
    • antenna ( plural antennae ) 触角
    • salamander 蝾螈
    • sea squirt 海鞘
  4. suspend

PAN = all

  1. panacea
    • alchemist 炼金术士
    • concoct 烹制
    • elixir 灵丹妙药
    • plague 困扰
  2. pandemonium 混乱
    • uproar 喧嚣
    • commotion 骚乱
    • goalie 守门员
    • herald 传令官
    • proclaim 宣布
    • solemn 庄严的
    • forthwith 立刻
  3. pantheism 泛神论
    • be identical with 与······相同
    • impersonal 平常无奇的
  4. panoply 华丽服饰；全副盔甲
    • foliage 叶子
    • stirring 激动人心的

EXTRA = outside / beyond

  1. extradite 引渡

  2. extrapolate 推断

  3. extrovert 外向的

  4. extraneous 无关的；外部的

PHOT = light

  1. photoelectric

  2. photovoltaic

  3. photon 光子

  4. photosynthesis 光合作用

LUC = light / to shine or glitter

  1. lucid
    • intact 完整的
  2. elucidate 阐明
    • jargon 术语
    • shed light on 提供线索；阐明
    • murky 不清白的
  3. lucubration （艰难的）研究
    • sarcasm 讽刺
  4. translucent 半透明的
    • goblet 高脚杯

MOR / MORT = to die / death

  1. mortality 死亡

  2. moribund 缺乏活力的；过时的

    • outmoded 过时的
    • lament 哀悼
  3. amortize
    • mortgage 抵押
  4. mortify
    • subdue 抑制
    • deaden 减弱
    • devout 虔诚的

TROPH = nourishment

  1. atrophy 萎缩
    • bedridden 卧床不起的

  2. hypertrophy 肥大

  3. dystrophy 营养障碍

  4. eutrophication 富营养化

    • algae 水藻
    • mollusk 软体动物
    • manure 粪肥
    • detergent 清洁剂

Words from Mythology and History

  1. aeolian harp 风弦琴
    • sinew 肌腱
  2. cynosure
    • constellation 星座
  3. laconic 谈吐简洁的
    • militaristic 军国主义的
    • utter 发出
  4. mnemonic 帮助记忆的
    • a treble clef 高音谱号
  5. platonic 柏拉图式的
    • sitcom = a situation comedy 情景喜剧

  6. sapphic 女同性恋的

  7. Socratic 苏格拉底式的

  8. solecism 语病；失礼

    • blunder 大错
    • etiquette 礼节
    • track 带入
    • rug 地毯
